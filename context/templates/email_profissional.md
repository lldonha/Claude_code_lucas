# Templates de Email Profissional

## Apresentação/Primeiro Contato

```
Assunto: [Contexto] - Apresentação

Prezado(a) [Nome],

Meu nome é Lucas, sou [profissão/cargo] e entro em contato para [motivo].

[Breve contexto de 2-3 linhas]

Gostaria de agendar uma conversa para [objetivo]. Tenho disponibilidade em:
- [Opção 1]
- [Opção 2]

Fico à disposição para mais informações.

Atenciosamente,
Lucas
[Contato]
```

## Follow-up

```
Assunto: Re: [Assunto anterior] - Follow-up

Prezado(a) [Nome],

Gostaria de fazer um acompanhamento sobre [assunto] que discutimos em [data].

[Status atual / Próximos passos]

Podemos alinhar [ação necessária]?

Atenciosamente,
Lucas
```

## Agradecimento

```
Assunto: Agradecimento - [Contexto]

Prezado(a) [Nome],

Agradeço [o que está agradecendo].

[Comentário adicional relevante]

[Próximos passos, se aplicável]

Atenciosamente,
Lucas
```

## Solicitação

```
Assunto: Solicitação - [O que precisa]

Prezado(a) [Nome],

Solicito [o que precisa] para [finalidade].

Detalhes necessários:
- [Item 1]
- [Item 2]

Prazo desejado: [data]

Fico à disposição para esclarecimentos.

Atenciosamente,
Lucas
```

## Envio de Documento

```
Assunto: [Tipo de Documento] - [Referência]

Prezado(a) [Nome],

Segue em anexo [documento] conforme [contexto].

Principais pontos:
- [Ponto 1]
- [Ponto 2]

[Instruções de ação, se necessário]

Fico à disposição para dúvidas.

Atenciosamente,
Lucas
```
