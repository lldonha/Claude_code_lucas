# Perícias Técnicas

Esta skill fornece templates e orientações para elaboração de documentos técnicos de engenharia.

## Quando Usar

- Elaborar laudos de avaliação de imóveis
- Criar relatórios de vistoria técnica
- Redigir pareceres técnicos
- Desenvolver memoriais de cálculo
- Documentar inspeções prediais

## Normas de Referência

- **NBR 14653-1** - Procedimentos gerais de avaliação
- **NBR 14653-2** - Imóveis urbanos
- **NBR 14653-3** - Imóveis rurais
- **NBR 13752** - Perícias de engenharia
- **NBR 16747** - Inspeção predial

## Boas Práticas

1. Sempre referenciar normas técnicas aplicáveis
2. Documentar metodologia utilizada
3. Incluir registro fotográfico completo
4. Manter imparcialidade técnica
5. Usar terminologia técnica adequada
6. Registrar ART/RRT do trabalho

## Exemplos de Uso
